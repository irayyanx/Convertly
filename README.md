# Convertly

Smart currency converter browser extension with Arabic support and built-in calculator.

![Firefox](https://img.shields.io/badge/Firefox-Addon-orange)
![Version](https://img.shields.io/badge/version-2.0-blue)
![License](https://img.shields.io/badge/license-MIT-green)

---

## Overview

Convertly is a lightweight browser extension that detects and converts currencies directly on any webpage.

Designed for shoppers, travelers, and developers who want fast currency conversions without leaving the page.

---

## Features

* 💱 Convert currencies directly on any webpage
* 🧮 Built-in math calculator (example: `50+20`)
* 🌍 Multi-currency conversion
* 🌐 Arabic RTL support
* 🌙 Light and Dark themes
* ⚡ Fast and lightweight
* 🔒 Privacy-friendly (no tracking)

---

## Installation

### Firefox

Install from Firefox Add-ons:

https://addons.mozilla.org/firefox/addon/convertly/

### Chrome

Coming soon to Chrome Web Store.

### Manual Installation

1. Download this repository
2. Open:

```
chrome://extensions
```

3. Enable **Developer Mode**
4. Click **Load unpacked**
5. Select the extension folder
